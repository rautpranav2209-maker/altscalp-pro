# AltScalp PRO 🚀

> Real-time crypto scalping signal dashboard with AI-powered analytics, paper trading, and risk management.

## ✨ Features

- **📊 Live Scanner** — 40+ crypto pairs with real-time signal scoring across 7 categories (Mid-cap, Meme, DeFi, AI, Gaming, L2, Infra)
- **📈 Advanced Charting** — Custom candlestick/bar/line charts with EMA, Bollinger Bands, RSI, MACD overlays
- **🤖 AI Signal Engine** — Multi-factor signal computation with SHAP explainability
- **💰 Paper Trading** — Full order management: market/limit/stop orders with TP/SL
- **⚡ Risk Management** — Circuit breaker, Kelly position sizer, daily loss caps
- **📱 Responsive Design** — Works on desktop, tablet, and mobile
- **🔔 Alert System** — Sound alerts, simulated Telegram integration
- **📊 Analytics** — Win rate tracking, equity curves, P&L heatmaps, signal decay analysis
- **🧠 AI/ML Dashboard** — Feature importance, anomaly detection, backtesting engine
- **💾 Persistent State** — Orders, trades, and settings saved to localStorage

## 🖥️ Pages

## ⌨️ Keyboard Shortcuts (Chart Page)

| Key | Action |
|-----|--------|
| `L` | Switch to Long |
| `S` | Switch to Short |
| `Enter` | Place order |
| `Escape` | Close modal |

## 🚀 Deploy

This is a **single HTML file** — no build step required.

### GitHub Pages
1. Fork this repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Your site will be live at `https://yourusername.github.io/altscalp-pro`

### Local
Just open `index.html` in your browser.
